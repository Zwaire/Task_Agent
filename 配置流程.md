之后直接启动服务器就OK，从仓库git到服务器上，创建虚拟环境，安装依赖

更新一下系统并安装虚拟环境工具

```
sudo apt update
sudo apt install python3-pip python3-venv
```

在对应的文件夹下面git项目代码

```
git clone https://github.com/Zwaire/Task_Agent.git
```

配置Qwen API

```
cd /www/smart_scheduler/backend 
cd core
nano llm_engine.py
```

在这里密钥的硬编码处修改，我也不知道为什么，在.env里面是不对的

```
dashscope.api_key = "sk-68xxxxxxxxxxxxx你的真实密钥"
```

修改完毕之后创建虚拟环境并安装依赖

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

进入后端启动服务

```
uvicorn main:app --host 0.0.0.0 --port 8000
```
